DESCRIPTION OF THE GAME:
-----------------------

# MAIN OBJECTIVE #

Nano is a 2D Puzzle Platformer where you conduct charge blocks and machinery to overcome puzzles while managing your battery life. Data logs and dynamic dialogues drive the narrative of the game while immersing the player in the puzzle platforming gameplay. 

# GAME NARRATIVE #

In the year 2052, humanity has been exiled from earth after an A.I. developed to enhance the efficiency of a nuclear silo gained sentience and turned rogue. 

Humans left behind N.A.N.O whose task is to infiltrate and disarm the rogue A.I. for humans to eventually return to Earth.

DEBUG/CHEAT KEYS:
----------------

GIVE GOLD - P
KILL ALL ENEMIES - Z
UNLOCK PROJECTILE SKILL #5 - 1
UNLOCK PROJECTILE SKILL #6 - 2
UNLOCK PROJECTILE SKILL #7 - 3
UNLOCKED PROJECTILE SKILL #8 - 4 
UNLOCK AOE SKILL #1 - 5
UNLOCK AOE SKILL #2 - 6
UNLOCK AOE SKILL #3 - 7
UNLOCK AOE SKILL #4 - 8

*For PROJECTILE SKILL #1-4 are the basic class skills for the four classes, hence those skills are not unlockable.
*After cheating the skills in, if you want to reset them you will have to restart the game. It is coded in this way because 4 skills is the maximum each player can have.
